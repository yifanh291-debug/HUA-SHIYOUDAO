# 公众号贴图生成器

HUA-SHIYOUDAO 是一个面向微信公众号运营的本地文字贴图生成器。打开网页，输入文案，就可以快速生成适合公众号推文、封面、日签、金句卡片的 PNG 图片。

## 功能特点

- 输入文案后实时生成文字图片
- 支持方图、竖图、公众号封面比例
- 支持多分类模板库和配色方案
- 内置 90 个原创模板
- 模板分类覆盖热门、引导、简约、清新、复古、商务、科技、教育、美食、旅行、卡通、产品、营销
- 支持纸感、亚麻、微粒、丝绸、纯净等高级纹理
- 支持上传自定义背景图，并调节图感和虚化
- 支持上传背景自由取景，拖动选择画面重点
- 支持拖拽文案位置
- 支持拖拽水印位置
- 支持调节水印大小、位置和显示状态
- 一键导出高清 PNG
- 纯前端实现，不需要服务器，不会上传你的图片和文案

## 快速开始

下载本项目后，直接双击打开 `index.html` 即可使用。

也可以在项目目录启动一个本地服务：

```bash
python -m http.server 4177 --bind 127.0.0.1
```

然后访问：

```text
http://127.0.0.1:4177/
```

## 使用方法

1. 在左侧输入公众号文案。
2. 选择模板分类、模板、比例和配色。
3. 在右侧调节字号、行距、边距、文案位置。
4. 选择背景纹理，或上传自己的背景图。
5. 设置水印内容、大小和位置。
6. 点击「导出 PNG」保存图片。

画布上方有「文案 / 水印」切换按钮：

- 选择「文案」后，可以直接拖动画布中的文案位置。
- 选择「水印」后，可以直接拖动画布中的水印位置。

## 部署到 GitHub Pages

1. 将本项目上传到 GitHub 仓库。
2. 打开仓库的 `Settings`。
3. 进入 `Pages`。
4. Source 选择 `Deploy from a branch`。
5. Branch 选择 `main`，目录选择 `/root`。
6. 保存后等待 GitHub 生成访问地址。

## 隐私说明

本工具是纯前端应用。文案、上传的背景图片、水印内容都只在你的浏览器中处理，不会上传到任何服务器。
