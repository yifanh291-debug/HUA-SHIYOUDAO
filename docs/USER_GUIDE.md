# 公众号贴图生成器使用说明书

## 1. 产品简介

公众号贴图生成器是一款适合公众号运营、自媒体编辑、内容创作者使用的文字图片生成工具。它的目标是减少重复排版工作，让你只需要输入文案，就能快速生成具备质感的文字贴图。

适用场景：

- 公众号文章内插图
- 推文金句卡片
- 日签海报
- 活动预热图
- 社群转发图
- 公众号封面辅助图

## 2. 打开方式

### 方式一：直接打开

双击 `index.html` 文件即可使用。

### 方式二：本地服务打开

在项目目录运行：

```bash
python -m http.server 4177 --bind 127.0.0.1
```

然后在浏览器打开：

```text
http://127.0.0.1:4177/
```

## 3. 基础操作

### 输入文案

在左侧「文案」输入框中输入要生成图片的文字。输入后，中间画布会实时更新。

### 选择模板

模板用于快速切换视觉风格。模板区参考公众号编辑器的运营场景，按分类组织，当前包含：

- 热门：清醒便签、主编金句、运营备忘、松弛日签
- 引导：顶部关注、阅读原文、点个在看、扫码关注
- 简约：极简留白、杂志标题、细线笔记
- 清新：春日清新、少女粉、海盐蓝
- 复古：旧报纸、新中式、港风标题
- 商务：黑金摘要、报告摘要、创始人说
- 科技：AI霓虹、云端科技、数据脉冲
- 教育：课程笔记、备考提醒、知识卡片
- 美食：茶饮上新、食谱小记
- 旅行：城市漫游、明信片
- 卡通：可爱气泡、漫画强调、萌宠日记
- 营销：活动促销、新品发布、限时福利

选择一个分类后，下方会显示该分类下的模板。点击模板即可切换当前画布风格。

### 选择比例

当前支持三种常用比例：

- 方图：适合文章插图、朋友圈转发
- 竖图：适合日签、长图卡片
- 封面：适合公众号封面比例参考

### 选择配色

配色会影响文字、点缀线、背景纸张等视觉元素。建议根据文章主题选择。

## 4. 文案排版

右侧「文字」区域可以调节：

- 字号
- 行距
- 边距
- 横向位置
- 纵向位置

也可以在画布上方选择「文案」，然后直接拖动画布中的文字。

建议：

- 金句类文案适合较大字号和较宽行距。
- 信息密度高的文案适合降低字号。
- 封面比例空间较小，尽量控制文案长度。

## 5. 背景与纹理

右侧「背景」区域可以选择纹理：

- 纸感：适合日签、手账、成长类内容
- 亚麻：适合生活方式、品牌感内容
- 微粒：适合观点、商业、知识类内容
- 丝绸：适合高级感、情绪价值、女性向内容
- 纯净：适合极简排版

### 上传背景图

点击「上传背景图」可以选择本地图片作为背景。上传后可调节：

- 图感：控制背景图融入强度
- 虚化：控制背景图模糊程度

为了保证文字可读性，建议背景图不要过于复杂；如果背景图较花，可以提高虚化或降低图感。

## 6. 水印设置

右侧「水印」区域可以设置：

- 水印文字
- 是否显示水印
- 水印大小
- 横向位置
- 纵向位置

也可以在画布上方选择「水印」，然后直接拖动画布中的水印。

建议：

- 公众号名称建议保持较小，不要抢正文。
- 如果图片用于社群传播，可以适当放大水印。
- 水印放在边角或留白区域，整体更干净。

## 7. 导出图片

点击「导出 PNG」或「保存图片」即可导出当前画布。

导出尺寸会根据当前比例自动生成：

- 方图：1080 x 1080
- 竖图：1080 x 1440
- 封面：900 x 383

## 8. 常见问题

### 为什么别人打不开我的 `127.0.0.1` 地址？

`127.0.0.1` 是你本机地址，只能在你自己的电脑访问。要分享给别人，可以把项目压缩包发给对方，或部署到 GitHub Pages。

### 上传的图片会不会传到服务器？

不会。本工具是纯前端应用，上传图片只在你的浏览器中处理。

### 导出的图片文字太挤怎么办？

可以尝试降低字号、增加行距、减少文案长度，或切换成竖图比例。

### 想做成公开网页怎么办？

把项目上传到 GitHub，然后开启 GitHub Pages 即可。

## 9. 推荐工作流

1. 先输入文案。
2. 选择比例。
3. 选择模板和配色。
4. 调整字号、行距、位置。
5. 选择纹理或上传背景图。
6. 设置水印。
7. 导出 PNG。
